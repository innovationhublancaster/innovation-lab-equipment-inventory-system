# innovation-lab-equipment-inventory-system
Generated from Innovation Flow: Innovation Lab Equipment &amp; Inventory System

## MVP Implementation
A lightweight offline-capable (browser localStorage) web app is implemented with these core workflows from `docs/spec.md`:
- Asset catalog with project/cost-center tagging
- Check-out/check-in with required condition photo and overdue tracking
- Reservation conflict detection with waitlisting
- Maintenance scheduling
- Procurement request intake
- Immutable audit log and JSON export

## Run
```bash
npm test
npm run start
```
Then open `http://127.0.0.1:4173`.
